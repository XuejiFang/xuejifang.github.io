实验一

0. 经典控制和现代控制

0.1 什么是控制？

为了克服干扰的影响，达到期望的目标而对被控对象中的某一个（某些）物理量（被控变量）进行的操作。

本课程重点：自动控制\(\rightarrow\) 智能控制

  1. 经典控制
    • 以传递函数为数学基础
    • 单输出、单输出线性定常系统
  2. 现代控制
    • 以状态空间为基础
    • 较高性能要求的多变量变参数系统的最优控制问题
  3. 智能控制（非线性多输入多输出系统）

1，2为传统控制方法

自动控制系统：被控对象+控制器

0.2 控制系统的分类

  • 按系统结构

    • 开环控制

      • 通过实验来确定烤面包时间
      • 控制律：\(Model\ y=f(x)\)

    • 闭环控制

      • 增加反馈环节
      1. 很好的控制带有变量的系统
      2. 对出现的问题进行补偿和调整
      • 检测偏差、纠正偏差

  • 按系统输入信号特征

    • 恒值控制
    • 随动控制

  • 按系统传递信号变化

    • 连续控制
    • 离散控制

  • 按系统的固有内在特性

    • 线性控制
      • 同时满足叠加性和均匀性的系统
      • 可以用线性微分方程描述
      • 线性时变系统、
    • 非线性控制
      • 不同时满足叠加性和 均匀性
      • 饱和和死区特性、间隙特性、

0.3 控制系统的基本要求

稳、准、快

瞬态响应

既要快，又要平稳

稳态响应

稳态误差尽量小

系统的传递函数

线性定常系统在零初始条件下微分方程中的输出量的拉氏变换与输入量的拉氏变换之比： \[ G(s)=\frac{Y(s)}{U(s)} \]

拉普拉斯变换\(F(s)=\int_0^\infty f(t)e^{-st}dt\)

  • 为了简化计算，变成线性加减

规定：以分母的最高阶次来确定系统的阶次

0.4 状态空间

  • 高阶微分方程通过选择适当的变量可以变为一阶微分方程组
  • 一阶方程组个数=变量的个数=高阶微分方程（传递函数）的阶数=独立的储能元件的个数
  • 变量的选择不是唯一的

1. 概论

1.1 智能控制的产生和发展

  1. 由于复杂性、非线性、时变性、不确定性和不完全性，无法获得被控对象的精准数学模型
  2. 控制过程无法用传统的数学模型来描述，即无法解决建模问题
  3. 线性化系统无法反映真实系统
  4. 无法完成复杂任务

1.2 智能控制的定义、特点和应用

二元论：是人工智能与自动控制的交叉

三元论：引入运筹学\(IC=AC\cap AI\cap OR\)

2.1 专家系统

2.1.1 概述

专家系统是一类包含知识和推理的智能计算机程序

先行者——Feigenbaum（费根鲍姆）

基本功能取决于它所含有的知识

发展历史

  1. 初创期

    高度专业化、专门问题求解能力强、结构功能不完整、移植性差、缺乏解释功能

  2. 成熟期

    单学科专业型专家系统、结构完整、功能全面、移植性好、具有推理解释功能、启发式规则、不精准推理、产生式规则、框架、语义网络表达知识、限定英语

  3. 发展期

    解释、预测、设计、规划、监视、修理、指导和控制

2.1.2 构成

知识库和推理机 （必考）

知识库

  1. 事实性知识
  2. 过程性知识（推理）

推理机

对专家知识进行加工

  1. 正向推理

  2. 反向推理

    提出假设的结论，寻找支持的证据，若证据存在，则假设成立

  3. 双向推理

人机接口

用户接口、专家接口

  1. 知识库

    知识表示、知识库管理系统

  2. 推理机

  3. 人机接口

    用户、专家

2.2 专家控制系统

2.2.1 概述

在未知环境下，仿效专家的经验，实现对系统的控制

2.2.2 基本原理

  1. 直接型专家控制器

    直接专家控制器用于取代常规控制器，直接控制生产过程或被控对象

  2. 间接型专家控制器

    器用于和常规控制器相结合

  • 优化型专家控制器
  • 适应型专家控制器
  • 协调型专家控制器
  • 组织型专家控制器

与专家系统的区别

专家控制引入了专家系统的思想，但与专家系统存在区别

  1. 专家系统能完成专门领域的功能，辅助用户决策；专家控制能进行独立的、实时的自动决策。专家控制比专家系统对可靠性和抗干扰性有着更高的要求
  2. 专家系统处于离线工作方式，而专家控制要求在线获取反馈信息，即要求在线工作方式

知识表示

专家控制将系统是为基于知识的系统

关键技术

  • 知识的表达方法；
  • 从传感器中识别和获取定量的控制信号；
  • 将定性知识转化为定量的控制信号；
  • 控制知识和控制规则的获取。

3. 模糊控制的理论基础

3.1 概述

用模糊来调和对立

Zadeh表示法 （必考）

  1. 不需要被控对象的数学模型

  4. 构造简单，易于软件实现
  5. 鲁棒性好，适应性好

3.2 模糊集合

3.2.1 集合和隶属度函数

集合

大写字母表示

元素

小写字母表示

论域\(U\)

被考虑对象的所有元素，

特征函数\(\mu_A(x)\) \[ \mu_A(x)= \left\{ \begin{array}[ll]\\ 1& x\in A\\ 0& x\notin A \end{array} \right. \] 隶属度函数\(\mu_A(x)\) \[ \mu_A(x): U\to [0,1]\\ \mu_A(x)= \left\{ \begin{array}\\ 1& x\in A\\ (0,1)& x属于A的程度\\ 0& x\notin A \end{array} \right. \]

\(x\) 的集合\(U\) 称为论域，\(\mu_A(x)\) 称为\(x\) 属于\(A\) 的隶属度

\(U\) 的模糊集合\(A: A=\{(x,\mu_A(x)|x\in X\}\)

模糊集合的表示

  1. Zadeh表示法 \[ \begin{array}[lll]\\ 离散:&A=&\frac{\mu_A(x_1)}{x_1}+\frac{\mu_A(x_2)}{x_2}+\cdots+\frac{\mu_A(x_n)}{x_n}\\ 连续:&A=&\int_U{\frac{\mu_A(x)}{x}} \end{array} \]

    隶属度为0，可省略

  2. 序偶表示法 \[ A=\{(x_1,\mu_A(x_1)),(x_2,\mu_A(x_2)),\cdots,(x_n,\mu_A(x_n))|x\in U\} \]

    隶属度为0，可省略

  3. 向量表示法 \[ A=[\mu_A(x_1)\ \mu_A(x_2)\ \cdots \ \mu_A(x_n)] \]

    隶属度为0，不可省略

其他术语

  • 台集台 \[ A_S=\{x|\mu_A(x)>0\} \]

  • \(\alpha\) 截集 \[ \begin{array}[lll]\\ 强\alpha截集&A_\alpha&=&\{x|\mu_A(x)>\alpha\}&\alpha\in[0,1)\\ 弱\alpha截集&A_alpha&=&\{x|\mu_A(x)\ge\alpha\}&\alpha\in(0,1] \end{array} \]

  • 正则模糊集合

    • 若\(\max_{x\in X}\mu_A(x)=1\)

  • 凸模糊集合

    • \[ \mu_A(\lambda x_1+(1-\lambda )x_2)\ge \min(\mu_A(x_1),\mu_A(x_2)) \]

  • 分界点

    • 使得\(\mu_A(x)=0.5\) 的点\(x\)

  • 单点模糊集合

    • 在论域中，若模糊集合的台集合仅为一个点，且该点的隶属度函数\(\mu_A(x)=1\)，则称\(A\)为单点模糊集合

3.2.2 模糊集合的运算

  1. 相等

    \(\forall x\in U,\mu_A(x)=\mu_B(x)\)

  2. 包含

    \(\forall x\in U,\mu_A(x)\le \mu_B(x)\)

  3. 空集

    \(\forall x\in U,\mu_A(x)=0\to A=\empty\)

  4. 并集

    \(\mu_C(x)=\mu_A(x)\cup \mu_B(x)=\max[\mu_A(x),\mu_B(x)]\to C=A\cup B\)

  5. 交集

    \(\mu_C(x)=\mu_A(x)\cap \mu_B(x)=\min[\mu_A(x),\mu_B(x)]\to C=A\cap B\)

  6. 补集（镜像）

    \(\overline{A}: \mu_A^-(\mu)=1-\mu_A(\mu)\)

运算法则

  1. \(A\cup A=A\\A\cap A=A\)

  2. \(A\cup B=B\cup A\\A\cap B=B\cap A\)

  3. \((A\cup B)\cup C=A\cup(B\cup C)\\(A\cap B)\cap C=A\cap(B\cap C)\)

  4. \(A\cup(A\cap B)=A\\A\cap(A\cup B)=A\)

  5. \(A\cup(B\cap C)=(A\cup B)\cap(A\cup C)\\A\cap(B\cup C)=(A\cap B)\cup(A\cap C)\)

  6. \(\overline{\overline{A}}=A\)

  7. \(\overline{A\cup B}=\overline{A}\cap \overline{B}\\\ \overline{A\cap B}=\overline{A}\cup \overline{B}\)

  8. \(A\cup E=E,A\cap E=E\\ A\cup \empty=A,A\cap \empty=\empty\)

普通集合中的互补律在模糊集合中不成立

3.4.2 模糊矩阵运算与模糊关系

  1. 相等

    \(a_{ij}=b_{ij}\to A=B\)

  2. 包含

    \(a_{ij}\le b_{ij}\to A\subseteq B\)

  3. 并运算

  4. 交运算

    \(c_{ij}=1-a_{ij}\to C=\overline{A}\)
